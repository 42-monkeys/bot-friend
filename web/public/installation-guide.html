Bot Installation Guide

We are sorry! :(

After the review process, the bots will be easily installable from the store of the platforms. Bear with us! For now you will have to follow this instructions

Slack

Go to Slack apps and click on Create New App

Go to OAuth & Permission Tab, where you can add these permission scopes:
channels:history
chat:write
chat:write
im:write
users:read
reactions:read

Click Install to Workspace and add it to your relax channel

Copy your Bot User OAuth Token

Go to your config and add your Bot User OAuth Token to the Slack API Token field, then select the channel where you want the bot to write!

Discord

Go to Discord Create Apps and create an app

In the OAuth2 tab, select the link URL Generator

In the SCOPES section, select bot.
In the BOT PERMISSION section select Send Messages and Read Message History

Copy the Generated url and paste it in the browser. You will be adding your bot!

Copy the Generated url and paste it in the browser. You will be adding your bot!

Go to your config and add your Client Secret to the Discord API Token field, then select the channel where you want the bot to write! (this is hard, you must go in the channel via browser and copy the last number in the toolbar)
